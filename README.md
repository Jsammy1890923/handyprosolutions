# 🌌 Overview

**HandyProSolutions** is a modern handyman services platform and educational home-repair resource built with:

- ⚡ Next.js
- 🎨 Tailwind CSS
- 📝 MDX-powered content workflows
- 📈 SEO-first architecture
- 📱 mobile-responsive UI systems

The platform combines:

✅ professional handyman service pages  
✅ localized SEO landing pages  
✅ homeowner-friendly DIY tutorials  
✅ booking and lead-generation workflows  
✅ scalable blog infrastructure  

into a single production-ready digital platform.

# 🧑‍🔧 Voice & Copywriting Standards

## ✅ Friendly, Clear, and Practical

Content should:

- avoid unnecessary jargon
- explain concepts in plain English
- focus on action-oriented guidance
- make repair tasks feel approachable

---

## 💬 Writing Style Rules

| Standard | Approach |
|---|---|
| Use second person | “You’ll need…” / “Here’s how you fix it…” |
| Start with action | “Fix a clogged sink in 3 easy steps” |
| Break into steps | Use numbered lists and checklists |
| Add practical callouts | Include tips, warnings, and quick wins |
| Optimize readability | Short paragraphs and skimmable sections |

---

## 💡 Example Tone Conversion

| Overly Technical | HandyProSolutions Style |
|---|---|
| “Apply a bonding agent to the joint fitting.” | “Add a little glue where the two pieces connect.” |
| “Utilize PPE during demolition.” | “Wear goggles — drywall dust gets everywhere.” |
| “An Allen key may be required.” | “Grab that tiny L-shaped wrench (Allen key).” |
