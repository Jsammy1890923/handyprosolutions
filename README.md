# 🛠️ HandyProSolutions

HandyProSolutions is a professional handyman service website and blog built with Next.js, Tailwind CSS, and MDX.

Designed for fast performance, mobile responsiveness, SEO optimization, and easy connection to a custom .com domain. Includes blog, booking, and service features — all deployable on Vercel + Render.


---

### 🎤 Brand Voice & Copywriting Guidelines

HandyProSolutions is built for everyday homeowners, renters, and first-time DIYers — so the voice must be:

#### ✅ Friendly, Clear, and Empowering

* Use **simple, jargon-free language**
* Talk to the user like a helpful neighbor, not a contractor
* Assume they’ve never picked up a wrench — but want to learn

#### 🛠️ Tone Examples

| Bad (Too Vague or Technical)                  | Good (Friendly, Clear)                                  |
| --------------------------------------------- | ------------------------------------------------------- |
| “Apply a bonding agent to the joint fitting.” | “Add a dab of glue where the two pieces snap together.” |
| “Utilize PPE during drywall demolition.”      | “Wear goggles — drywall dust stings like crazy.”        |
| “An Allen key may be required.”               | “Grab that tiny L-shaped wrench (aka Allen key).”       |

#### 💬 Writing Style Rules

* **Use second person**: "You’ll need...", "Here’s how you fix it..."
* **Start with action**: “Fix a clogged sink in 3 steps”
* **Break into steps**: Use bullet lists or numbered steps whenever possible
* **Add tips/callouts**:

  > 💡 Pro Tip: Wrap teflon tape *clockwise* so it doesn’t peel off when screwing it in.

### 🧑‍🔧 Copy Goals

| Priority           | Guidelines                                                                                             |
| ------------------ | ------------------------------------------------------------------------------------------------------ |
| ✅ Trustworthy Help | Offer useful, practical info — not filler or hype                                                      |
| ✅ Visual Learning  | Use diagrams, step lists, images where possible                                                        |
| ✅ SEO-Optimized    | Naturally include long-tail keywords like “how to patch drywall” or “fix leaky faucet without plumber” |
| ✅ Localized Ready  | Can adapt to zip code pages (e.g., “Handyman in Marietta, GA”)                                         |
